# 🤖 Basic Chatbot

## 📌 Description
This project is a simple rule-based chatbot developed in Python as part of the CodeAlpha Python Programming Internship.

The chatbot interacts with the user through the command line and provides predefined responses to common greetings and questions.

## ✨ Features
- Greets the user
- Responds to "Hello"
- Answers "How are you?"
- Tells its name
- Responds to "Good Morning"
- Replies to "Thank You"
- Ends the conversation when the user types "bye"
- Handles unknown inputs with a default response

## 🛠 Technologies Used
- Python 3

## 📂 Project Files
- codealpha_basic_chatbot.py
- README.md

## ▶️ How to Run

1. Install Python 3.
2. Download this repository.
3. Open the project folder.
4. Run:

```bash
python codealpha_basic_chatbot.py
```

## 💬 Sample Conversation

```
🤖 Welcome to CodeBot

You: hello
Bot: Hi! 👋

You: how are you
Bot: I am fine, thanks! 😊

You: what is your name
Bot: My name is CodeBot.

You: thank you
Bot: You're Welcome! 😊

You: bye
Bot: Goodbye! Have a Nice Day. 👋
```
